## 微信端操作
### 微信端 seller 登录之后核销优惠卷
 - /api/cus/writeOffCoupons
 - POST
    - token 绑定成为 seller 的微信用户登录之后得到的 token
    - code  需要核销的优惠卷的 code
 - RES
 
    {
        "code": 1000,
        "msg": "write off coupon success",
        "data": null
    }

## 电脑端操作

    
###  销售网页端核销优惠卷
 - /api/sell/writeOffCoupons
 - POST
    - token 网页端后台 seller 登录之后获取到的 token
    - code  需要核销的优惠卷的 code
 - RES
 
    {
        "code": 1000,
        "msg": "write off coupon success",
        "data": null
    }
