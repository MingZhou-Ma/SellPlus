
### 增加销售
 - /api/mer/addSeller
 - POST
    - token   
    - Seller seller      销售对象
 - 返回
    
        {
            "code": 1000,
            "msg": "add seller success",
            "data": {
                "id": 1111,
                "pic": "xxxx"
                ...
            }
            
        }
