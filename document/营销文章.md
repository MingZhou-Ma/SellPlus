### 增加阅读历史
 - /api/cus/read
 - POST
    - token
    - articleid 文章id
 - 返回
        
        {
            "code": 1000,
            "msg": "save readHistory success",
            "data": null
        }
 
## 后台
### 增加阅读文章
 - /api/mer/addArticle
 - POST
     - token   后台商家用户登录后的 token
     - title   标题
     - content 文章内容
     - author  作者
     - createDate  创建时间 (yyyy-MM-dd hh:mm:ss)
 
 - RES
        
        {
            "code": 1000,
            "msg": "xxxxxxxxxx",
            "data": null
        }
 
 
### 删除文章
 - /api/mer/delArticle
 - POST
    - token
    - articleid 文章 id
 
 - RES
 
        {
            "code": 1000,
            "msg": "xxxxxxxxxx",
            "data": null
        }
